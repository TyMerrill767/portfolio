All About Me

Hello, and welcome to my site! I’m Tyler Merrill, a web developer, meaning I handle all the coding and programming of a website. This is where you’ll be able to see my work, or contact me if you have a question, comment, or have some work for me. I’m proficient in HTML, CSS, using frameworks such as Bootstrap, JavaScript, and Each and every website I build is designed to be accessible – young or old, tech expert or technologically impaired, even the disabled. Day by day, site by site, I’m going to do my best to make the web more beautiful. Come on in!
